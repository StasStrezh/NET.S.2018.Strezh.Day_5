# Day 5
1) (TaskFindGCD)Разработать класс, позволяющий выполнять вычисления НОД по алгоритму Евклида для двух, трех и т.д. целых чисел. Добавить к разработанному классу методы, реализующие алгоритм Стейна (бинарный алгоритм Евклида) для расчета НОД двух, трех и т.д. целых чисел. Разработать unit-тесты для тестирования методов данного типа. (NUnit фреймворк)
2) (ConvertingToOtherSystem)Реализовать метод расширения получения из строкового представления целого положительного четырехбайтового числа, записанного в p-ичной системе счисления (2<=p<=16), его десятичного значения. Разработать модульные тесты. (NUnit фреймворк).
